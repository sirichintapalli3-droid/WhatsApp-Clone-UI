# WhatsApp Clone UI 💬

A simple WhatsApp Web clone built using **HTML, CSS, and JavaScript**.  
This project replicates the basic UI and chat functionality for learning and practice purposes.

---

## 🚀 Features

- 📱 WhatsApp-like responsive UI
- 👤 Contact list with profile pictures
- 🔍 Search bar to filter chats
- 💬 Chat open on click
- 🟢 Online status UI design
- ⚙️ Settings / profile section (UI only)
- 📂 Clean folder structure (HTML, CSS, JS separated)

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
